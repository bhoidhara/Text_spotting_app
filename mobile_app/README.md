# VisionText Mobile (Flutter)

## Setup

1. Install Flutter SDK.
2. In this folder, run:
   ```bash
   flutter pub get
   ```
3. Run on emulator/device:
   ```bash
   flutter run --dart-define=API_BASE_URL=https://your-render-url.onrender.com
   ```

## Notes
- This app uses `X-User-Id` header for demo auth.
- Replace the API base URL with your Render URL.
